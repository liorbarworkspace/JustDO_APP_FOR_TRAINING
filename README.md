## The Problem

Most fitness apps assume you have a gym. A trainer. A fixed schedule. Expensive equipment.

Most people don't.

They have 30–60 minutes, a pair of dumbbells, a mat, and the motivation to get better — if someone just helped them channel it.

**JUSTDO is that someone.**

---

## What It Is

JUSTDO is a fully personalized home workout system. Not a content library. Not a generic program. A tool that lets you — or any user — build, manage, and execute a training plan tailored to their body, their equipment, and their goals.

The app runs entirely in the browser. No account required, no subscription, no friction. Open it, start your onboarding, and train.

---

## Core Features

| Feature | Description |
|---|---|
| **Exercise Library** | Build your own catalog — name, equipment, muscle groups, safety notes, level |
| **Workout Templates** | Compose reusable sessions from your library |
| **Weekly Planner** | Assign templates to days, switch plans, manage multiple programs |
| **Live Session** | Real-time timer, audio cues, set/rep tracking, mid-session edits |
| **AI Plan Generator** | Describe your goals → get a full weekly plan generated by Gemini AI |
| **Personal AI Coach** | Chat with an AI coach for guidance, tips, and exercise suggestions |
| **Progress Tracker** | Log completions, view history, export data as CSV |
| **Dark Mode** | Full light/dark theme support |
| **CSV Import / Export** | Move exercises, templates, and plans in and out — complete data portability |

## Technology Choices

JUSTDO is built with a deliberate, lean stack — every choice has a reason.

**React 19 + TypeScript** — Component-driven UI with full type safety. The app is complex (20+ components, deep state) — TypeScript makes it maintainable as it grows.

**Vite** — Near-instant dev builds. Production bundles are small and fast.

**Tailwind CSS** — Utility-first styling with dark mode built in. No CSS files to manage.

**Google Gemini API** — Powers the AI Plan Generator and Personal Coach. Chosen for quality, speed, and free-tier access — lowering the barrier for users and future deployment.

**localStorage + React state** — No backend, no database, no server costs. All data stays in the user's browser. This is a conscious product decision: zero infrastructure means zero cost to run and instant global deployment via any static host.

## Getting Started

### Prerequisites

- Node.js 18+
- A free [Google Gemini API key](https://aistudio.google.com/app/apikey) _(only needed for AI features)_

### Run Locally

```bash
# Clone
git clone https://github.com/liorbarworkspace/JustDO_APP_FOR_TRAINING.git
cd JustDO_APP_FOR_TRAINING

# Install
npm install

# Set up environment
cp .env.local.example .env.local
# Edit .env.local and add your key:
# GEMINI_API_KEY=your_key_here

# Start
npm run dev
```

App runs at `http://localhost:5173`

### Build for Production

```bash
npm run build
npm run preview
```

The output is a static site — deploy to Vercel, Netlify, GitHub Pages, or any CDN.

## Environment Variables

```env
# .env.local
GEMINI_API_KEY=your_gemini_api_key_here
```

The AI features (Plan Generator, Personal Coach) require a Gemini API key.  
All other features — workout management, session tracking, progress history — work without it.
